# Actuarial Risk Pricing Environment

An OpenEnv environment that simulates an insurance market. The AI agent must calculate base premiums, loading factors, and reinsurance percentages. The goal is to maximize total profit while maintaining a safe solvency ratio (>1.5) to avoid penalties. 

## Quick Start (Demo)
To test the environment, update the API keys and run:
`python client.py --scenario user`
